Mission Planner

Description

This app is design to serve in planning missions of an industrial robot. It empowers operators to create and visualize robot movement.
The user can add the points of interest that the robot should reach ( 2D points) in a specific order. Then he can control the movement of the robot between those points.

Structure

The app contains three pages:

  • Home page: the current page. It contains information about the application and how you can use it.
  • Planner page: where you can add, remove and manage the points which the robot will reach during the mission.
  • Player page: where you can visualize and control the robot's mission .
